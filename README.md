# math-react

## Getting Started

### Development
```sh
$ yarn || $ npm i 

$ yarn start || $ npm run start
```

## folder
### actions 
- API 호출 함수 관리
- `superagent` 라이브러리 사용

### components
- `view`를 관리하는 파일 폴더

### containers
- `store`와 `view`를 연결해주는 파일 폴더
- `store`의 action 을 수행하기전 데이터 처리 수행


### stores
- `mobx` 라이브러리 사용

- init from [react-mobx-boilerplate](https://github.com/kkangil/react-mobx-boilerplate)
